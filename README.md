# aws-starter-project

## AWS EC2 Web Server + S3 Backup + CloudWatch Monitoring

This project demonstrates deploying a web server on EC2, automating backups to S3 using Bash, and monitoring the instance using CloudWatch metrics and alarms.

It uses: 
- EC2 (Compute)  
- S3 (Storage & Backups)  
- CloudWatch (Monitoring)  
- Bash Scripting  
- IAM Roles  
- VPC Networking (Free tier)
